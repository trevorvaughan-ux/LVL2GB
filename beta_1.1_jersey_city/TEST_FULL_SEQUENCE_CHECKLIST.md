# Test the Full Low-Pressure Sequence — Jersey City Beta 1.1

Use this checklist to test the complete flow on real schools in the current beta (39 public + 10 private).

## Philosophy Reminder
- Zero pressure.
- Once someone engages (opens or replies), they move to Community.
- Goal = give value and stay top of mind.
- Small CTAs only.

## Daily / Per-Reply Test Flow

1. **Get a reply** (or simulate one for testing)
2. **Sync it into the sheet** (run "Sync Recent Replies from Supabase")
3. **See it in Action Needed tab**
4. **Decide priority**
   - Hot / worth a call soon → mark Priority = Yes
   - Warm / future → mark as "Warm / Future Opportunity"
   - Already engaged → move to Community
5. **Take the next small action**
   - Schedule a call/meet (log in Notes or Next Action)
   - Send value (Friday Moment, breathing video, Science of Calm deck, Hanuman preview, etc.)
   - Log the value touch in "Last Value Touch" + date
6. **Follow up lightly** (small CTA button in the value content)
7. **If no buy** → keep in Community and continue occasional value touches

## Columns to Use for This Test

- Priority (Yes / Warm / No)
- Status (Community, Warm / Future Opportunity, In Conversation, Call Booked, Materials Sent, etc.)
- Last Value Touch
- Last Value Touch Date
- Next Action
- Notes

## Success for Today

You can receive a reply (real or test), see it clearly in the sheet, move it to the right status, and log a value touch — all without the data feeling scattered or risky.

This is the smallest real test of the full machine for the current beta.

Once this feels smooth on the current 49 schools, we can expand the same pattern to the next districts (Manhattan, etc.).
